Sobre mí

Mi trayectoria, mi enfoque y lo que me mueve

Llegué al desarrollo web por curiosidad y me quedé por convicción. Cuando desarrollo una aplicación no pienso únicamente en que funcione, sino en que tenga sentido para la persona que la utiliza.

Me hago algunas preguntas clave durante el proceso:

¿Será intuitiva?

¿La entenderá alguien que nunca la ha usado?

¿Qué ocurre si el usuario comete un error?

¿Cómo puedo reducir la frustración y mejorar la experiencia?

Mi formación en Antropología me enseñó a observar las necesidades humanas; el código me dio las herramientas para transformarlas en soluciones reales.

Antes de dedicarme al código, trabajé durante años en hostelería. Allí aprendí a gestionar la presión en entornos exigentes, coordinar equipos y resolver problemas con agilidad en contacto directo con personas.

He construido dos aplicaciones completas desde cero, liderando un equipo en mi Trabajo Fin de Grado y desarrollando un sistema de alquiler de bicicletas con gestión de clientes, reservas, generación de contratos PDF y envío automático de emails .

Esta experiencia me ha enseñado que la tecnología, cuando se hace con empatía, no solo funciona mejor, sino que también tiene impacto real.

Busco un entorno donde pueda seguir aprendiendo, aportar mi visión y demostrar que el código bien hecho, con una mirada humana, es una herramienta poderosa.

Sistema de alquiler de bicicletas

Proyecto Fullstack

Rol: Fullstack Developer (Frontend + Backend)

Problema

Las pequeñas empresas de alquiler necesitaban un sistema digital que permitiera gestionar reservas, disponibilidad y contratos sin procesos manuales ni errores administrativos.

Solución

Diseñé y desarrollé una aplicación fullstack que digitaliza todo el flujo de alquiler: desde la creación de clientes hasta la generación automática de contratos y notificaciones por email.

Impacto funcional

  • ✔ Gestión completa de clientes, bicicletas y reservas
  • ✔ Generación automática de contratos PDF
  • ✔ Sistema de disponibilidad por fechas en tiempo real
  • ✔ Notificaciones automáticas por email
  • ✔ Control de roles y autenticación JWT

Stack técnico

Angular Spring Boot PostgreSQL JWT Flyway JavaMail

Arquitectura

Arquitectura en capas: Controller → Service → Repository. API REST desacoplada con frontend Angular modular. Gestión de entidades normalizadas en PostgreSQL.

ShowSync | Plataforma de gestión de eventos

Fullstack System · TFG

Rol: Fullstack Developer · Team Lead (4 personas)

Problema

La gestión de eventos, artistas y espacios suele realizarse con herramientas dispersas (Excel, emails, agendas manuales), lo que provoca errores de coordinación, conflictos de calendario y pérdida de información.

Objetivo del sistema

Diseñar una plataforma centralizada que permitiera gestionar eventos, artistas y recursos en tiempo real, con control de disponibilidad, roles de usuario y comunicación estructurada entre equipos.

Solución técnica

Desarrollamos una aplicación fullstack basada en arquitectura cliente-servidor con Angular y Spring Boot, implementando un sistema de gestión de calendario, autenticación segura y control de disponibilidad dinámico.

Responsabilidades clave

  • ✔ Liderazgo de equipo de 4 desarrolladores
  • ✔ Diseño de arquitectura backend (Spring Boot)
  • ✔ Implementación de sistema de reservas y calendario
  • ✔ Gestión de roles: administrador, promotor y artista
  • ✔ Integración frontend-backend y consumo de API REST

Impacto técnico

  • ✔ Sistema de disponibilidad por fechas en tiempo real
  • ✔ Autenticación segura con JWT
  • ✔ Gestión estructurada de entidades relacionales
  • ✔ API REST con separación por capas (Controller / Service / Repository)
  • ✔ Escalabilidad para múltiples tipos de eventos y recursos

Stack tecnológico

Angular 19 Spring Boot PostgreSQL JWT Flyway JPA/Hibernate REST API

Arquitectura

Arquitectura en capas backend (Controller → Service → Repository) + frontend modular en Angular. Base de datos relacional normalizada con control de disponibilidad por entidad y fecha.
