Sobre mí

Mi trayectoria, mi enfoque y lo que me mueve

Llegué al desarrollo web por curiosidad y me quedé por convicción. Mi formación en Antropología me enseñó a observar las necesidades humanas; el código me dio las herramientas para resolverlas.

Antes de dedicarme al código, trabajé en hostelería durante años. Allí aprendí a gestionar la presión en entornos de ritmo exigente, a coordinar equipos con precisión y a resolver problemas de cara al público con agilidad. Esa experiencia me enseñó que la tecnología, como el servicio al cliente, tiene que ser eficaz, clara y estar pensada para quien la usa.

He construido dos aplicaciones completas desde cero, liderando un equipo en mi TFG y asumiendo el desarrollo integral de un sistema de alquiler de bicicletas con gestión de clientes, reservas, generación de contratos y envío automático de emails. Esta experiencia me ha enseñado que la tecnología, cuando se hace con empatía, transforma.

Busco un entorno donde pueda seguir aprendiendo, aportar mi visión y demostrar que el código bien hecho, con una mirada humana, es una herramienta poderosa.

Proyectos destacados

Mi trabajo en acción: aplicaciones completas con propósito

Sistema de alquiler de bicicletas

Aplicación fullstack completa para gestión de alquiler de bicicletas con sistema de reservas, generación de contratos PDF y notificaciones automáticas.

Mi enfoque: Desarrollé este proyecto de forma autónoma, desde el diseño de la base de datos hasta la implementación del frontend. Mi objetivo era crear una herramienta funcional, fácil de usar y con un flujo de trabajo claro para el usuario final.

Angular Spring Boot PostgreSQL Flyway API REST JavaMail

Características principales:

  • CRUD completo de clientes y bicicletas
  • Sistema de reservas y disponibilidad
  • Generación automática de contratos PDF
  • Envío de emails de confirmación
  • Autenticación y roles de usuario

ShowSync | Gestión de Eventos

Trabajo de Fin de Grado (TFG): Aplicación fullstack para la gestión integral de eventos, shows y artistas. Permite coordinar calendarios, recursos y contrataciones de manera eficiente.

Mi rol: Lideré un equipo de 4 personas, coordinando tareas, repartiendo trabajo y asegurando la calidad del proyecto. Aprendí a gestionar la complejidad de un sistema grande y a tomar decisiones técnicas que priorizaran la experiencia del usuario.

Angular Spring Boot PostgreSQL API REST JWT JPA/Hibernate

Características principales:

  • Gestión completa de artistas, shows y espacios
  • Sistema de reservas y disponibilidad de calendario

  • Autenticación JWT y roles de usuario
  • Panel de administración avanzado
